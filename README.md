# CJ_Collection
Welcome to CJ_Collection!

Here you can find educational material, examples, videos and helpful links.  

All materials, links and etc was chosen by me.  

Its a collection of some links, videos, books and etc that helped me a lot, gave a better understanding, another point of view  or had a really nice and easy explanation.

Provided material is not sponsored, if it will be sponsored or provided by someone it will be mentioned in the comments.  
More material will be added over time.
